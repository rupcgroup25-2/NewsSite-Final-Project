About MYNews

Your personalized news experience powered by cutting-edge technology

MYNews is a revolutionary news platform that combines artificial intelligence, real-time data processing, and user-centric design to deliver the most relevant and engaging news experience tailored just for you.

Advanced Features

Cutting-edge functionality that goes beyond traditional news websites

Advanced Text-to-Speech

Multi-voice English TTS with pause/resume functionality, word highlighting, and voice selection (US, UK, AU)

Real-time Chat System

Firebase-powered chat for each article with real-time messaging, user authentication, and message persistence

Interactive World Map

JQVMap integration with country selection and real-time Twitter trending hashtags by location via Twitter API

Push Notifications

Firebase Cloud Messaging with service worker for background notifications and automatic token management

AI-Powered Features

HuggingFace API for article summarization and AI-generated profile images with multiple AI service integration

Social Profile System

Complete user profiles with following system, activity tracking, and Cloudinary image management

Advanced Archive Search

NewsApi API integration for comprehensive archive search with date range filtering and advanced search UI

Real-time Admin Dashboard

Comprehensive analytics, user management, report handling, and real-time statistics with advanced controls

Advanced UI/UX

Bootstrap 5 dark/light mode, responsive design, CSS animations, and progressive enhancement

Technical Implementation Highlights

Full-Stack Architecture

ASP.NET Core backend with Firebase integration, jQuery frontend, and SQL Server database

Security Features

JWT authentication, role-based access control, input validation, and XSS protection

Performance Optimization

Lazy loading, image optimization, browser caching, and stored procedure optimization

Modular Design

Three-tier architecture, shared components, and feature-based organization
